Bi-Pedal Robot Modeling & Simulation

Designing and optimizing an MPC controller for a bipedal robot across walking, running, stair climbing, and obstacle courses — all in MuJoCo.

MuJoCo MPC Controls Simulation Robot Dynamics Optimization
🤖

Getting a bipedal robot to walk is solved. Getting it to walk, run, climb stairs, and navigate obstacles — all with one controller — is not. This project tackled that generalization problem using Model Predictive Control in MuJoCo, treating each locomotion mode as a different constraint set rather than a different controller entirely.
