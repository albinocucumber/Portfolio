Bio-Inspired Adaptive Fin for Enhanced Underwater Propulsion

Designing fins that flex like a fish — and building a test rig precise enough to actually prove it works.

Bio-Inspired Design Fabrication Test Rig Design Load Sensing Structural Analysis Fluid Dynamics
🐟

Fish have solved underwater propulsion problems that engineers are still working on. A tuna can accelerate faster than most AUVs, using a flexible tail that actively changes stiffness mid-stroke. This project started with a simple question: can we capture that behavior in a manufacturable, testable mechanical system?
